# Zerodha
A full-stack Zerodha trading platform clone built with React, Node.js, Express, and MongoDB, featuring portfolio management, trading dashboard, and stock tracking.
# Zerodha Clone – Full Stack Trading Platform

This project is a **full-stack clone of the Zerodha trading platform** built to simulate the core functionality of an online stock trading application. It demonstrates how modern trading dashboards work and provides hands-on experience with full-stack web development.

## 🚀 Features

* User authentication (login & signup)
* Interactive trading dashboard
* Portfolio and holdings management
* Watchlist for tracking stocks
* Buy/Sell order simulation
* Fund management
* Responsive UI

## 🛠 Tech Stack

### Frontend

* HTML
* CSS
* JavaScript
* React.js

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/chandana02-29/Zerodha.git
```

2. Navigate into the project folder

```
cd Zerodha
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```

## 🎯 Purpose

The goal of this project is to understand how **modern trading platforms like Zerodha work** and to practice building scalable full-stack web applications.
