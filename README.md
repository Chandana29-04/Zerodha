#Zerodha Clone – Full Stack Trading Platform

A full-stack clone of the Zerodha trading platform built using React, Node.js, Express, and MongoDB.  
This project simulates a stock trading dashboard with portfolio management, order tracking, and market watchlist features.
## 🚀 Features

* User authentication (login & signup)
* Interactive trading dashboard
* Portfolio and holdings management
* Watchlist for tracking stocks
* Buy/Sell order simulation
* Fund management
* Responsive UI

## 🛠 Tech Stack

### Frontend

* HTML
* CSS
* JavaScript
* React.js

### Backend

* Node.js
* Express.js

### Database

* MongoDB

```
Zerodha/

└── README.md
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/chandana02-29/Zerodha.git
```

2. Navigate into the project folder

```
cd Zerodha
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```

## 🎯 Purpose

The goal of this project is to understand how **modern trading platforms like Zerodha work** and to practice building scalable full-stack web applications.
